網路掃描工具

針對目標域名或 IP 執行 DNS 查詢、WHOIS 查詢、HTTP 標頭分析與 Port 掃描。 適用於資安研究、滲透測試前期偵查與網路診斷。

本工具僅限對您擁有或已獲得明確授權的目標使用。未經授權掃描第三方系統可能違法。掃描結果由第三方 API 提供，僅供參考。

DNS 查詢

解析 A、MX、NS、TXT 等記錄，了解域名的 IP 位址與郵件伺服器配置。

WHOIS

查詢域名或 IP 的註冊資訊，包含擁有者、註冊商、到期日與聯絡資訊。

Port 掃描

偵測目標主機開放的常用 TCP 連接埠，協助識別暴露的服務與潛在攻擊面。
